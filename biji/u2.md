### unsigned

1. **只能存储非负数（0 或正数）**。

2. **能表示的最大值比有符号整数大一倍**，因为不需要存储负数。

3. **默认整数是 `signed`（有符号的）**，需要手动加 `unsigned` 才能变成无符号数。

   

###  `unsigned` vs `signed` 选择建议

| **情况**           | **选择 `signed`（默认）** | **选择 `unsigned`**     |
| ------------------ | ------------------------- | ----------------------- |
| 需要负数           | ✅                         | ❌                       |
| 需要更大的数值范围 | ❌                         | ✅                       |
| 可能会溢出         | ❌（有 `int` 溢出检测）    | ❌（无符号溢出不会警告） |
| 计数、大小、索引   | ❌（可能出错）             | ✅                       |

📌 **一般情况下，优先使用 `signed`，除非明确不需要负数！**

### 取余运算

#### **1. 取余运算规则**

- **只能用于整数**（`int`、`long`等），不能直接用于浮点数（`float`、`double`）。
- **余数的符号与被除数相同**（与 C 语言标准兼容）。
- **公式**： a%b=a−(a/b)×ba \% b = a - (a / b) \times ba%b=a−(a/b)×b 其中 `a / b` 取整后再相乘，最终得到 `a % b` 的值。
- printf("%d\n", 10 % 3);   // 1，因为 10 / 3 = 3 余 1    
- printf("%d\n", -10 % 3);  // -1，余数的符号与被除数相同    
- printf("%d\n", 10 % -3);  // 1，余数的符号与被除数相同    
- printf("%d\n", -10 % -3); // -1，余数的符号与被除数相同

#### **2. 取余运算的应用**

### 4，预防错误

scanf 函数的返回值是成功读取并赋值的输入项数。
在你的代码中，scanf(" <%d><%c>", &n, &T) 期望读取两个输入项：
一个整数（%d），存储到变量 n 中。
一个字符（%c），存储到变量 T 中。
因此，如果 scanf 成功读取并赋值了这两个输入项，它将返回 2。如果返回值不是 2，说明输入格式不正确，可能是因为输入的不是整数或者字符，或者输入的格式不符合预期。

异或操作（XOR）是一种位运算符，用符号 ^ 表示。它的运算规则是：如果两个对应的二进制位相同，则结果为 0；如果不同，则结果为 1。

### 在数组中找到唯一的数

#include <stdio.h>

int findUnique(int arr[], int size) {
    int res = 0;  // 初始化结果为 0
    for (int i = 0; i < size; i++) {
        res ^= arr[i];  // 逐个元素异或
    }
    return res;  // 返回唯一的数
}

# **C 语言数组详细讲解**

数组（Array）是 **一组相同类型的元素** 在内存中 **连续存储** 的数据结构。在 C 语言中，数组是非常重要的数据类型，广泛用于存储和管理多个相同类型的值。

------

## **1. 数组的基本概念**

### **1.1 什么是数组？**

数组是一组相同类型的数据元素，**存放在连续的内存空间中**，通过索引（下标）访问。

### **1.2 数组的特点**

- **元素类型相同**（只能存储同一种数据类型）。
- **内存连续存储**（所有元素在内存中是挨着的）。
- **索引（下标）访问**（从 `0` 开始编号）。
- **大小固定**（定义时必须指定长度，无法动态扩展）。

------

## **2. 一维数组**

### **2.1 定义数组**

```c
类型 数组名[大小];
```

示例：

```c
int numbers[5];  // 定义一个包含 5 个整数的数组
```

这里：

- `int` 说明数组元素是整数类型。
- `numbers` 是数组名称。
- `[5]` 表示数组可以存放 5 个整数。

------

### **2.2 数组的初始化**

数组可以在声明时赋初值：

```c
int numbers[5] = {1, 2, 3, 4, 5};  // 初始化数组
```

也可以部分初始化，未赋值的元素默认为 `0`：

```c
int numbers[5] = {1, 2};  // 剩余元素默认为 0
// 等价于 {1, 2, 0, 0, 0}
```

如果初始化时提供了全部数据，可以省略大小：

```c
int numbers[] = {1, 2, 3, 4, 5};  // 自动推导大小为 5
```

------

### **2.3 访问数组元素**

数组索引从 `0` 开始，可以使用 `数组名[索引]` 访问元素：

```c
int numbers[5] = {10, 20, 30, 40, 50};
printf("%d\n", numbers[0]);  // 输出 10
printf("%d\n", numbers[3]);  // 输出 40
```

------

### **2.4 修改数组元素**

```c
int numbers[3] = {1, 2, 3};
numbers[1] = 100;  // 修改索引 1 位置的值
printf("%d\n", numbers[1]);  // 输出 100
```

------

### **2.5 遍历数组**

用 **for 循环** 遍历数组：

```c
#include <stdio.h>

int main() {
    int numbers[5] = {10, 20, 30, 40, 50};

    for (int i = 0; i < 5; i++) {
        printf("%d ", numbers[i]);
    }

    return 0;
}
```

**输出**

```
10 20 30 40 50
```

### **3.4 遍历二维数组**

```c
#include <stdio.h>

int main() {
    int matrix[2][3] = {
        {1, 2, 3},
        {4, 5, 6}
    };

    for (int i = 0; i < 2; i++) {
        for (int j = 0; j < 3; j++) {
            printf("%d ", matrix[i][j]);
        }
        printf("\n");
    }

    return 0;
}
```

**输出**

```

4 5 6
```

## **4. 数组与指针**

数组变量本质上是指向 **首元素的指针**。

### **4.1 数组名是首地址**

```c
int numbers[3] = {10, 20, 30};
printf("%p\n", numbers);      // 地址
printf("%p\n", &numbers[0]);  // 地址（相同）
```

### **4.2 指针访问数组**

```c
int numbers[3] = {10, 20, 30};
int *p = numbers;  // 指针指向数组
printf("%d\n", *p);      // 10
printf("%d\n", *(p + 1));  // 20
```
